# procdata

Processed intermediates live here. The pipeline assumes generated metadata under `procdata/metadata/`, prepared profile tables under model-specific subdirectories, and other intermediate assay inputs as needed.

Observed usage after the latest default run with CPCNN and CellProfiler is about 7.8 GB. Most of this is processed assay data, including the CellProfiler scalar morphology features. Files here are ignored by git.
